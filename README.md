
  # Snap&Sort App Design

  This is a code bundle for Snap&Sort App Design. The original project is available at https://www.figma.com/design/WNyvu6PY8MPDS9oPJ0V7jg/Snap-Sort-App-Design.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Environment

  Copy `.env.example` to `.env` and set your keys. Required for auth and stats: Supabase URL and anon key from the [Supabase dashboard](https://supabase.com/dashboard) (Settings → API). Optional: Gemini, Groq, and Google Maps keys — see comments in `.env.example`.
